Connecting Data

CMAC-Pipeline software provides a simplified path to publishing NASA Obs4MIPs data to the Earth System Grid Federation.

Download 0.2.0
Installation Documentation

Participants

This effort involves the close collaboration of several key stakeholders, including the Goddard Earth Sciences Data and Information Services Center, at the NASA Goddard Space Flight Center (GSFC), the Physical Oceanography DAAC at the Jet Propulsion Laboratory (JPL), the Atmospheric Science Data Center at the NASA Langley Research Center (LaRC), and the Earth System Grid Federation.

The software is principally developed by members of the Data Management Systems and Technology group at the Jet Propulsion Laboratory.

Documentation

Documentation for the software includes detailed Installation instructions as well as a collaborative wiki providing in-depth architectural and system details. Send an email to cmac-dev@jpl.nasa.gov if you desire write access to the wiki.

About This Site

This site provides software developed in response to the CMAC12: Next Generation CyberInfrastructure call. More information about the NASA solicitation that this effort responds to can be found at the NASA NSPIRES website.

For technical issues or questions related to the installation or use of CMAC software, please send an email to cmac-dev@jpl.nasa.gov
